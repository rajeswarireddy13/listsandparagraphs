Lists in HTML:

What are lists? Lists are used to group a set of related items in no particular order(unordered lists), in a specific order(ordered lists), display name/value pairs such as terms and definitions(description lists)

For Example:

An unordered HTML List:

  • Column
  • Column
  • Column
  • Column
  • Column

An ordered HTML List:

  1. Column 1
  2. Column 2
  3. Column 3
  4. Column 4
  5. Column 5
